-什么是node.js
    +javascript 运行时既不是一个语言，也不是框架，他是一个平台
-node.js中的javascript包括：
    +没有BOM和DOM
    +EcmaScript基本的JavaScript语言部分
    +在node.js中为JavaScript提供了一些服务器级别的API
        *文件操作能力
        *http服务能力

- ip地址和端口号的理解

    +端口号用来定位具体的应用程序
